# Video Production Agent

Full-stack AI video production system that automatically researches topics, generates professional multi-angle videos with narration, source overlays, and captions.

## Features

- **Automated Research**: Web search via Perplexity or Gemini with source extraction
- **Multi-Angle Videos**: 1-6 camera perspectives with seamless transitions
- **Native Voice Audio**: Generated by Veo 3.0 alongside video
- **Source Citations**: Screenshot overlays from research sources
- **Synchronized Captions**: Auto-timed subtitles that split text into phrases
- **Parallel Generation**: Video clips generate concurrently for faster production
- **Job Persistence**: Productions survive backend restarts
- **Web UI**: React frontend for monitoring and playback


## Quick Start

### 1. Install Dependencies

```bash
cd ElectOMate-Backend/scripts/video_gen_tests/video_agent

# Backend
pip install -r requirements.txt

# Frontend
cd frontend
npm install
cd ..
```

### 2. Set Environment Variables

```bash
# Required
export GEMINI_API_KEY="your-key-from-aistudio.google.com"

# Optional (for better web search)
export PERPLEXITY_API_KEY="your-perplexity-key"
```

### 3. Start Services

```bash
# One-command startup (recommended)
./start.sh

# Or start manually:
# Terminal 1 - Backend
cd backend && uvicorn main:app --host 0.0.0.0 --port 5322 --reload

# Terminal 2 - Frontend
cd frontend && npm run dev -- --port 5321
```

### 4. Open Browser

Navigate to: **http://localhost:5321**

## Usage

### Via Web UI

1. Open http://localhost:5321
2. Configure production:
   - Topic/Prompt
   - Language (English, Hungarian, German)
   - Style/Tone (serious, casual, funny, dramatic)
   - Duration (30-120 seconds)
   - Number of perspectives (1-6)
   - Screenshot frequency (never, rare, moderate, frequent, always)
3. Click "Produce Video"
4. Monitor progress (updates every 3 seconds)
5. Watch video in-browser or download when complete

### Via API

**Submit Production:**
```bash
curl -X POST http://localhost:5322/api/produce \
  -H "Content-Type: application/json" \
  -d '{
    "topic": "Democracy in Hungary",
    "language": "English",
    "tone": "serious",
    "duration_seconds": 60,
    "num_perspectives": 3,
    "orientation": "landscape",
    "resolution": "720p",
    "screenshot_frequency": "moderate",
    "generate_captions": true
  }'
```

**Response:**
```json
{
  "job_id": "democracy_in_hungary_20260405_0515",
  "status": "pending",
  "message": "Production queued"
}
```

**Monitor Progress:**
```bash
curl http://localhost:5322/api/jobs/democracy_in_hungary_20260405_0515
```

**Download Video:**
```bash
# Regular video
curl http://localhost:5322/api/jobs/democracy_in_hungary_20260405_0515/video -o video.mp4

# With captions
curl http://localhost:5322/api/jobs/democracy_in_hungary_20260405_0515/video?captioned=true -o video_captioned.mp4
```

## Configuration Options

| Parameter | Type | Options | Default | Description |
|-----------|------|---------|---------|-------------|
| `topic` | string | any | (required) | What the video is about |
| `language` | string | English, Hungarian, German | English | Script language |
| `tone` | string | serious, casual, funny, dramatic | serious | Narration style |
| `duration_seconds` | int | 30-120 | 60 | Total video length |
| `num_perspectives` | int | 1-6 | 3 | Camera angles/characters |
| `orientation` | string | landscape, portrait | landscape | 16:9 or 9:16 format |
| `resolution` | string | 720p, 1080p | 720p | Output quality |
| `character_gender` | string | male, female | male | Character voice/appearance |
| `character_age` | string | any | 40 | Character age descriptor |
| `character_description` | string | any | (auto) | Override character prompt |
| `search_sources` | array | web, knowledge_base, wikipedia | ["web"] | Research sources |
| `screenshot_frequency` | string | never, rare, moderate, frequent, always | moderate | How often to show sources |
| `generate_captions` | bool | true, false | true | Burn subtitles into video |
| `background_music_path` | string | file path | "" | Optional MP3 for background |
| `music_volume` | float | 0.0-1.0 | 0.12 | Music volume level |

### Screenshot Frequency Guide

- **never** — No source overlays
- **rare** — 20% of segments show sources
- **moderate** — 50% of segments show sources (default)
- **frequent** — 80% of segments show sources
- **always** — Every segment that requests a source shows it

## API Reference

### Production Endpoints

#### `POST /api/produce`
Submit a new production request.

**Request Body:** `ProductionRequest` schema (see Configuration Options)

**Response:**
```json
{
  "job_id": "string",
  "status": "pending",
  "message": "Production queued"
}
```

#### `GET /api/jobs`
List all productions (persisted from disk on startup).

**Response:**
```json
[
  {
    "job_id": "democracy_in_hungary_20260405_0515",
    "status": "completed",
    "step": "done",
    "message": "Loaded from disk",
    "topic": "Democracy in Hungary"
  },
  ...
]
```

#### `GET /api/jobs/{job_id}`
Get job status and result.

**Response:**
```json
{
  "job_id": "string",
  "status": "pending|running|completed|failed",
  "step": "string",
  "message": "string",
  "result": {
    "final_video": "/path/to/video.mp4",
    "final_video_captioned": "/path/to/video_captioned.mp4",
    "metadata_path": "/path/to/metadata.json",
    "sources": [...],
    "duration_seconds": 56.0
  }
}
```

#### `GET /api/jobs/{job_id}/video`
Download or stream video file.

**Query Parameters:**
- `captioned` (bool, optional) — If true, returns captioned version

**Response:** video/mp4 file

#### `GET /api/jobs/{job_id}/metadata`
Get full production metadata.

**Response:** JSON metadata (see Metadata Structure)

#### `GET /api/jobs/{job_id}/progress`
Get detailed progress log.

**Response:**
```json
{
  "job_id": "string",
  "status": "string",
  "progress_log": [
    {
      "step": "research",
      "message": "Researching topic...",
      "level": "STEP"
    },
    ...
  ]
}
```

## Cost Estimates

Production costs depend on video length and complexity:

| Component | Per Unit | 60s Video | Notes |
|-----------|----------|-----------|-------|
| Gemini 2.5 Flash (research + script) | $0.30/1M tokens | ~$0.004 | Text generation |
| Imagen 3 (character images) | ~$0.04/image | ~$0.24 | 6 angles |
| **Veo 3.0 (video clips)** | $0.10-0.20/sec | **$5.60-$11.20** | Main cost |
| Google TTS | $16/1M chars | ~$0.03 | Narration |
| ffmpeg processing | Free | $0 | Local |
| **Total** | — | **$7-9** | Per 60s video |

### Cost Reduction Tips

1. **Reduce duration**: 30s video = ~$4-5 (half the Veo cost)
2. **Fewer perspectives**: 1 angle instead of 3 = -66% image cost
3. **Lower screenshot frequency**: "rare" or "never" = faster processing
4. **Batch productions**: Generate multiple variations of same topic

## Production Time

Typical production timeline for a 60-second video:

| Phase | Duration | Parallel |
|-------|----------|----------|
| Research | 20-30s | No |
| Script generation | 10-15s | No |
| Character images | 30-60s | Yes (all angles) |
| Video clips (7 clips) | 5-7 min | Yes (4 concurrent) |
| Post-processing | 20-40s | No |
| **Total** | **6-9 minutes** | — |

## Logging

The agent logs everything to multiple formats:

1. **stdout** — Colored terminal output with timestamps
2. **agent_log.txt** — Human-readable text log
3. **agent_log.jsonl** — Structured JSON log (one entry per line)

All tool calls, inputs, outputs, and timings are captured.

Example log entry:
```json
{
  "level": "TOOL_CALL",
  "step": "generate_video_clip",
  "message": "Calling generate_video_clip",
  "details": {
    "inputs": {
      "prompt": "A 40-year-old male professional...",
      "duration": 8
    }
  },
  "elapsed": "2:34",
  "timestamp": "2026-04-05T03:17:45Z"
}
```

## Troubleshooting

### "No module named 'google.genai'"
```bash
pip install google-genai
```

### "GEMINI_API_KEY not set"
```bash
export GEMINI_API_KEY="your-key"
# Or add to .env file in project root
```

### "Research returned 0 sources"
This happens when using Gemini fallback (no Perplexity key). The agent will still generate placeholder source cards. To get real web screenshots:
```bash
export PERPLEXITY_API_KEY="your-key"
```

### "ffmpeg not found"
```bash
# macOS
brew install ffmpeg

# Ubuntu/Debian
sudo apt-get install ffmpeg
```

### Captions not syncing properly
This was fixed in commit `3886e3a`. Update to latest version:
```bash
git pull origin hungarian-knowledgegraph
```

### Backend restart loses jobs
Job persistence was added in commit `73026a4`. Jobs are now loaded from `output/*/production_metadata.json` on startup.

### Video player not showing in UI
In-browser video players were added in commit `69e043a`. Make sure frontend is rebuilt:
```bash
cd frontend
npm install
npm run dev
```

### Adding New Features

1. **Research sources**: Extend `agent/tools/research.py`
2. **Video styles**: Modify `agent/tools/video_gen.py` character templates
3. **Post-processing**: Add effects in `agent/tools/post_process.py`
4. **API endpoints**: Add routes in `backend/main.py`
5. **UI controls**: Update `frontend/src/App.tsx`
